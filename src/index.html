Principales technologies utilisés :

Le language principal que j'utilise pour le back-end est le PHP, pour sa structuration, et le fait qu'il est le plus utilisé pour le back-end sur le web.

En système de gestion de base de données j'utilise principalement mySql, et pour la connexion depuis le php je préfère mysqli_connect à PDO.

En structuration du code PHP j'utilise principalement la programmation orienté objet, qui semble plus claire et lisible, ainsi que plus flexible et modulable.

Pour structurer les gros projets, je préfère les structurés avec une architecture MVC, qui permet de séparer le code en plusieurs parties et de faciliter la modification et la maintenance.
